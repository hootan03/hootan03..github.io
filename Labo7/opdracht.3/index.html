De generator voorziet de cellen in deze rijen van een andere achtergrondkleur, zoek op waar dit precies staat in de gegenereerde CSS. Waarom heeft de volgende CSS regel geen effect als je deze toepast op een cel in een rij met class="alt", maar wel in een rij zonder deze 'alt' class? .alt is specifieker dan onbekend in de specificity rangorde, dus voorop geplaatst.
